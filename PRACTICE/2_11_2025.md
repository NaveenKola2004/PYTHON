### 1.Find Factorial of a number

def Factorial(n):
    result=1
    if n==1 or n==0:
        return 1
    for i in range (1,n+1):
        result=result*i
    return result
n=int(input("Enter the number : "))

### 2.Count number of digits in a number
- This one counting the number of digits in an given number 
- ex :- 123 -> 3

```python
def Count_Digit_Number(n):
    count=0
    while (n>0):
        count+=1
        n=n//10
    return count
n=int(input("Enter the number : "))
print(Count_Digit_Number(n))
```
