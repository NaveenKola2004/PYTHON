```python
Given a string, check if it is a palindrome when you:

Ignore spaces

Ignore punctuation/symbols

Ignore case (treat A == a)

Example:

"A man, a plan, a canal: Panama" ‚Üí ‚úÖ palindrome

"race a car" ‚Üí ‚ùå not palindrome
```

### Example

```python
def ispoli(s):
    left=0
    right=len(s)-1
    while left<right:
        while left<right and not s[left].isalnum():
            left+=1
        while left<right and not s[right].isalnum():
            right-=1
        if s[left].lower()!=s[right].lower():
            return False
        left+=1
        right-=1
    return True
print(ispoli("A man, a plan, a canal: Panama"))
```
