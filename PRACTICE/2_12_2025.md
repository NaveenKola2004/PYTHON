# Problem 1 Emirp Number

#### what it is

```python
An Emirp is a number that is:

Prime

When reversed â†’ STILL prime

But NOT a palindrome

```


```python
Example:

13 â†’ reverse 31 â†’ both prime â†’ âœ” Emirp
17 â†’ 71 â†’ âœ”
11 â†’ reverse 11 â†’ palindrome â†’ âŒ Not Emirp
```

### Hear is the sample code in python ðŸ‘‡ðŸ‘‡

```python
def Prime(n):
    for i in range(2,n):
        if n%i==0:
            return False
    return True
def Reverse(n):
    rev=0
    while n>0:
        digit=n%10
        rev=(rev*10)+digit
        n//=10
    return rev
def Poli(n):
    rev=Reverse(n)
    if rev==n:
        return True
    else:
        return False
def Solution(n):
    prime_Check=Prime(n)
    if prime_Check:
        revrse=Reverse(n)
        if Prime(revrse):
            if Poli(n):
                return "not emirp"
            else:
                return "emirp"
        else:
            return "not emirp"
    else:
        return "not emirp"
n=int(input())
print(Solution(n))
```

# Problem 2  Second Largest Element Without Sorting

```python
Example:

List: [12, 35, 1, 10, 34, 1]
Answer â†’ 34
```

### Hear is the sample code in python ðŸ‘‡ðŸ‘‡

```python
def Second_largest(n):
    if len(n)<2:
        return "Need one more number"
    largest=float("-inf")
    sec_largest=float("-inf")

    for num in n:
        if num>largest:
            sec_largest=largest
            largest=num
        elif num > sec_largest and num!=largest:
            sec_largest=num
    return sec_largest
print(Second_largest([12, 35, 1, 10, 34, 1]))
```
