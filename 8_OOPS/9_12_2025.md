### 1. Encapsulation (hide data, control access)

- Python doesn’t force encapsulation, but it supports it.
#### public,protected,private
```python
class details:
    def __init__(self,id,name,mark):
        self.id=id                  # public
        self._name=name             # protected
        self.__mark=mark            # private
    def Show(self):
        print(self.id)
k=details(1,"naveen",45)
print(k.id)
print(k._name)
print(k.__mark)
```

### 2. Inheritance (child gets parents’ traits)

```python
class Animal:
    def sound(self,name):
        self._name=name
        print("Animal sound",name)
class Dog(Animal):
    def brak(self,name):
        self._name=name
        print("Bow Bow",name)
k=Dog()
k.sound("kola")
k.brak("naveen")
```
